SPLUNK EDGE PROCESSOR
=====================

Congratulations! You have successfully deployed the Splunk Edge Processor to your Kubernetes cluster.

Verify your deployment by running these commands:

1. Verify the Helm chart deployment (shows the installed Splunk Edge Processor release):
   helm list -n {{ .Release.Namespace }}

2. List the Kubernetes service and available ports:
   kubectl get svc -n {{ .Release.Namespace }} -l "app.kubernetes.io/instance={{ .Release.Name }}"
   kubectl get svc {{ .Release.Name }}-edge-processor -n {{ .Release.Namespace }} -o wide

3. Check the status of your deployment:
   kubectl get pods -n {{ .Release.Namespace }} -l "app.kubernetes.io/instance={{ .Release.Name }}"

4. View the logs for troubleshooting:
   kubectl logs -n {{ .Release.Namespace }} -l "app.kubernetes.io/instance={{ .Release.Name }}"

5. View deployment details:
   kubectl describe statefulset {{ .Release.Name }}-edge-processor -n {{ .Release.Namespace }}

6. Check persistent volumes (if configured):
   kubectl get pvc -n {{ .Release.Namespace }}
